What is this class?
Equipment is a simple POJO (Plain Old Java Object) representing a piece of equipment in a HobbyKit. It demonstrates basic encapsulation with private fields and public getters/setters.

üìò Definition: POJO (Plain Old Java Object)

A POJO is a simple Java object that:

  • Has private fields
  • Has public getters and setters
  • Has a constructor to initialize fields
  • May override toString()
  • Doesn't extend special classes or implement framework interfaces
