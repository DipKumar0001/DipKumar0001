ğŸ“˜ Singleton Pattern

Category: Creational Design Pattern

Intent: Ensure a class has only ONE instance and provide a global access point to it.

Implementation Requirements:

  • Private static instance variable
  • Private constructor (prevents external instantiation)
  • Public static getInstance() method

static means there's only ONE instance variable shared across all references.

The private constructor prevents using new BookingSystem() from outside.

Lines 33-38: getInstance() - Lazy Initialization

public static BookingSystem getInstance() {
    if (instance == null) {
        instance = new BookingSystem();
    }
    return instance;
}

Lazy Initialization: Instance is created only when first requested, not at class loading time.

ğŸ“˜ What are Java Streams?

Streams provide a functional approach to processing collections:

  • stream() - creates a stream from a collection
  • filter() - keeps elements matching a condition
  • map() - transforms each element
  • collect() - gathers results into a collection
  • findFirst() - returns first matching element

findUserById() with Streams

public User findUserById(String userId) {
    return users.stream()
            .filter(user -> user.getUserId().equals(userId))
            .findFirst()
            .orElse(null);
}

Lambda Expression: user -> user.getUserId().equals(userId) is a function passed to filter().

getAvailableItems() with Method Reference

public List<BookableItem> getAvailableItems() {
    return items.stream()
            .filter(BookableItem::isAvailable)  // Method reference
            .collect(Collectors.toList());
}

BookableItem::isAvailable is a method reference - shorthand for item -> item.isAvailable()

Sorting with Stream

users.stream()
    .sorted((u1, u2) -> Integer.compare(u2.getLoyaltyPoints(), u1.getLoyaltyPoints()))
    .forEach(user -> System.out.println(user.getFullName()));

Sorts users by loyalty points in descending order.
